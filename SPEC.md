# Wedding Landing Page - SPEC.md

## Project Overview
- **Name:** Nico & Franziska - Wedding Landing Page
- **Type:** Simple, elegant informational website for wedding guests
- **Core functionality:** Display key information about the wedding day
- **Target users:** Wedding guests

## Design Vision
Elegant, romantic, and modern. Soft color palette with warm tones. Clean typography with a touch of sophistication. The page should feel personal and inviting, not corporate.

## Layout & Structure
- **Hero Section:** Names of the couple with wedding date and romantic tagline
- **When & Where:** Ceremony and reception details (date, time, venue, address)
- **Our Story:** Brief intro about the couple (optional section)
- **RSVP Section:** Simple call-to-action for guest responses
- **Footer:** Contact info and social links

## Features & Interactions
- Smooth scroll navigation
- Elegant animations on scroll (subtle fade-ins)
- RSVP button that opens mailto or external form
- Mobile responsive design

## Technical Approach
- Single HTML file with embedded CSS and minimal JS
- Vanilla HTML/CSS/JS (no frameworks needed for this scope)
- Google Fonts for elegant typography
- Soft color scheme: cream, sage green, gold accents
